# Traditional Tamil Wedding Invitation

An elegant, cinematic, and fully local Tamil wedding invitation website.

## Features
- Vanilla HTML/CSS/JS (No frameworks needed)
- Fully responsive on all devices
- Smooth scroll & Parallax effects
- Glassmorphism & Gold Shimmer effects
- Traditional Tamil elements (Kolam, Kuthu Vilakku, Jasmine flowers)
- LocalStorage RSVP system
- Background music controls

## Folder Structure
- `index.html` - The main website
- `style.css` - All styling, variables, animations
- `script.js` - Interactions, countdown, audio controls, RSVP logic
- `assets/` - Directory containing `img/` and `audio/`

## How to Run Locally
1. Simply double-click on `index.html` to open it in your browser.
2. No backend or server is required. Everything runs locally in your browser.

## Customization
- **Text & Details:** Open `index.html` and search for the text you want to change (e.g., names, dates, venues).
- **Images:** Replace the placeholder images in the `assets/img/` folder with your own. Ensure you keep the same filenames, or update the paths in `index.html`.
- **Music:** Replace `assets/audio/bgm.mp3` with your preferred track.
- **Colors:** Open `style.css` and modify the CSS variables at the top of the file under `:root`.

## How to Deploy on GitHub Pages
1. Create a new repository on GitHub.
2. Upload all files from this folder to the repository.
3. Go to Repository Settings > Pages.
4. Select the `main` branch and `/root` folder, then save.
5. Your website will be live at `https://[your-username].github.io/[repo-name]`.
